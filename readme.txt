== Description ==
= Awesome Logos WordPress Plugin =

Ideal to showcase logos of sponsors, clients.

Show Image portfolios with Grid having optional external links, external images by Image URL.

Grid can be of two types: adjustable adaptable to available space or responsive Grid.

= Features: =

1. Lightbox for images.
2. Simple Shortcode.
3. Hover Effects
4. Slider with swiping.
5. Widget Ready
6. Categories Logos/Images
7. 38 customization options for Slider
8. 17 customization options for Grid
9. 15+ Arrows and Bullet skins for Slider
10. Multiple Sliders/Grids on single page.
11. Title with overlay or CSS Tooltip
12. Option to Add, Delete, Reorder Slides.
[See live demo](http://www.wpshopee.com/awesome-logos-wordpress-plugin/)

== Installation ==
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Awesome Logos** and click "*Install now*"
1. Alternatively, download the plugin and upload the contents of 'awesome-logos.zip' to your plugins directory, which usually is '/wp-content/plugins/'.
1. Activate the plugin
